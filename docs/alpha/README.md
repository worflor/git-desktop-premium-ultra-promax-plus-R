# Alpha Hardening Pack

This directory contains the required hardening documents for external alpha readiness.

Contents:
- onboarding.md: first-run setup and validation checklist
- known-issues.md: user-facing issue list with workarounds
- security-privacy-review.md: security and privacy posture summary
- feedback-triage-cadence.md: feedback intake and prioritization process

Maintenance rule:
- Update these documents in every alpha patch cycle where behavior, risks, or required setup changes.
